# SWP391 Jira & GitHub Integration Tool

## Description
A supporting tool for requirements and project progress management in Software Project course (SWP391),
integrating Jira Software Cloud and GitHub.

## Team
- Backend: Nguyễn Đức Minh (SE184433), Phạm	Tuấn Anh (SE192861), Nguyễn	Thành Minh (SE194125)
- Frontend: Nguyễn Phùng Đại Lộc (SE192238)

## Technologies (Proposed)
- Frontend: ReactJS
- Backend: Java Spring Boot
- Database: SQL Server
- Integration: Jira Cloud REST API, GitHub REST API
- Containerization: Docker, Docker Compose
- Testing & API Tools: Postman

## Repository Structure
- /backend   : Spring Boot REST API
- /frontend  : ReactJS Web Application
- /docs      : Project documents (SRS, diagrams, demo scripts)

## Development Process
- Git Flow with main / frontend / backend branches
- Jira for requirements and task management
- GitHub for source control
