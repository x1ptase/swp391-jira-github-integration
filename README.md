# SWP391 Jira GitHub Tool Support

## Description
A supporting tool for requirements and project progress management in Software Project course (SWP391), integrating Jira Software Cloud and GitHub.

- Integration: Jira Cloud REST API, GitHub REST API
- Testing & API Tools: Postman, Swagger UI

## Repository Structure
- /backend   : Spring Boot REST API
- /frontend  : ReactJS Web Application

## Development Process
- Jira for requirements and task management
- GitHub for source control
